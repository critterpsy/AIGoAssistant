# ADR-003: Style Policy and Human Plausibility

## Decision

La fuerza adaptativa y la personalidad de juego se implementan mediante reponderacion
dentro de un conjunto aceptable de jugadas, usando quality floor y plausibility gate.

## Consequences

- el rival no se vuelve arbitrariamente torpe;
- el estilo y el rank objetivo afectan distribuciones, no solo ruido.
